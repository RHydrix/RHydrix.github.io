Basic File Format

Anything inside of two // is personal to you anything inside of || is technically optional, anything within two ~~ is required and anything inside of ## is where you would name a variable or function (this can be anything!)

Top of file

/Name/ /GitHub Link/ This is commented out

/simple explanation/ This is commented out

|Requirements: this is where you would indicate any required npm modules or files|

e.g. `const #variableName# = require('#Modual or File#');`

function #functionName#(){

Your code goes here

~return~

}

~module.exports = {#functionName#};~

This is required for wafflesbot to access it
